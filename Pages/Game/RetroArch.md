### Load Core

## Settings

### Input

#### Hot Keys

- Menu Toggle: Key f1
- Fast-Forward (Hold): Key l
- Rewind: Key r
- Load State: Key f4
- Save State: Key f2
- Show FPS (Toggle): Key f3

#### Hot Key Configurations

Device Index: Controller (XBOX 360 For Windows)

- L2 Button (Trigger): Key f4
- L3 Button (Thumb): Key r
- R3 Button (Thumb): Key f2

### Frame Throttle

#### Rewind

- Rewind Support: On
- Rewind Frames: 5
